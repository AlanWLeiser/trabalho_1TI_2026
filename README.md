# O Seu "Top 5" !
### Descrição do Projeto:
- Chegou a hora de mostrar para o mundo o que você curte! Seu objetivo é criar um artigo web, estilo Buzzfeed, listando o seu "Top 5" sobre qualquer assunto que você seja fã: seus 5 jogos favoritos, as 5 melhores séries da Netflix, os 5 melhores jogadores do seu time, ou até os 5 melhores lanches da cantina.

- Neste projeto, vamos focar em construir "caixas" (Box Model) para separar bem o conteúdo, usando bordas, espaçamentos internos e externos para criar um visual organizado e agradável.

## O que deve ser feito (Estrutura da Página):

  - Cabeçalho: Um título gigante (`<h1>`) com o tema do seu Top 5 e uma imagem principal de destaque logo abaixo.

  - A Lista (5 seções): Você deve criar 5 blocos de conteúdo. Para cada item do seu Top 5 (do 5º até o 1º lugar), você precisará incluir:

  - Um subtítulo (`<h2>` ou `<h3>`) com o nome e a posição do item.

- Uma imagem ilustrativa.

- Um parágrafo (`<p>`) explicando o motivo daquela escolha.

- Uma linha separadora (`<hr>`) ao final do bloco (opcional, dependendo do seu design).

- Rodapé: Um parágrafo pequeno no final da página com os seus créditos (Ex: "Criado por [Seu Nome] - Turma X").

## O que deve ser usado (Regras Técnicas):

- HTML: Utilize corretamente as tags de estruturação (h1 a h6, p, img, br, hr).

- CSS **Inline** Exclusivo: Todo o estilo deve ser feito usando o atributo style="..." diretamente nas tags do HTML (como vimos na Semana 9 do nosso curso). 

> Atenção: Vai dar trabalho copiar e colar estilos repetidos? Sim! E é de propósito para valorizarmos a próxima matéria do curso!

- Estilos Obrigatórios:
  - Cores (color, background-color).
  - Fontes (font-size, text-align, font-weight).
  - Imagens controladas (width, height).
  - Limites da caixa (border e border-radius para arredondar cantos).

> O Desafio: Usar padding (para dar "respiro" entre o texto e a borda da caixa) e margin (para separar uma caixa da outra).

## O que deve ser entregue:

- Um link para seu repositório público do github:

  - O seu arquivo principal chamado index.html.

  - Um a pasta chamada imagens com todas as fotos usadas no projeto (não use links diretos da web para as imagens, baixe-as e organize na pasta).
